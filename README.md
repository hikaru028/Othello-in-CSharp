# Realtime 2Players OthelloGame in C#


## Set up
1. In the terminal, navigate to the directory containing your Program.cs file.
```
cd Server
```

2. Build the program:
```
dotnet build
```

3. Run the programme:
```
dotnet run
```

It should show like `Listening at 127.0.0.1:8080`.

4. Open client/wwwroot/`index.html` file
